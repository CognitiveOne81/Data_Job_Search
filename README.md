# Facebook Marketplace Dashboard (Search: `mac`)

A lightweight dashboard that displays the first 10 matching Facebook Marketplace-style items with **name + price**.

## Run

```bash
python app.py
```

Then open: `http://localhost:8000`

## Notes

- The dashboard defaults to the query `mac`.
- Results are loaded from a local fixture (`data/sample_marketplace_mac.json`) and filtered server-side.
